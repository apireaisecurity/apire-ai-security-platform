# Phase 3: Product Suite Expansion Plan

This document outlines the plan for Phase 3, which involves the creation of three major containerized web applications for the APIRE AI Security Platform.

## 1. APIRE Prompt Shield

**Web-Based Prompt Testing Framework**

- **Goal**: Real-time injection attack testing and jailbreak detection.
- **Stack**: Node.js/Express, React, PostgreSQL, Redis.
- **Status**: Specification created in `docs/specs/prompt-shield/SPECIFICATION.md`.

## 2. APIRE RedTeam Kit

**Web-Based AI Red Team Toolkit**

- **Goal**: Advanced adversarial testing and scenario-based attacks.
- **Stack**: NestJS, Next.js, MongoDB, RabbitMQ.
- **Status**: Specification created in `docs/specs/redteam-kit/SPECIFICATION.md`.

## 3. APIRE Compliance Checker

**Web-Based Regulatory Scanner**

- **Goal**: Automated policy scanning for GDPR, HIPAA, EU AI Act, etc.
- **Stack**: Node.js/Express, Vue.js, PostgreSQL, Elasticsearch.
- **Status**: Specification created in `docs/specs/compliance-checker/SPECIFICATION.md`.

## Next Steps

1.  Review specifications.
2.  Initialize repositories or monorepo workspaces for each tool.
3.  Begin implementation of "Prompt Shield" MVP.
