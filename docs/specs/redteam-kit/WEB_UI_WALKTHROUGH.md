# Web UI Walkthrough

## Dashboard
- **Active Campaigns**: See what's running now.
- **Vulnerability Trend**: Line chart showing security posture over time.
- **Recent Findings**: List of latest successful attacks.

## Attack Library
- Browse 50+ scenarios.
- Filter by "Injection", "Leakage", "Toxicity".
- Click "Run" to launch a quick test against a configured target.

## Campaign Planner
- **Targets**: Define your endpoints (URL, Headers, Auth).
- **Schedule**: Set up nightly or weekly scans.
- **Scenarios**: Select a subset of attacks to run.

## Report Generator
1.  Select a Campaign.
2.  Choose Template (Executive Summary, Technical Detail).
3.  Click **Generate PDF**.
