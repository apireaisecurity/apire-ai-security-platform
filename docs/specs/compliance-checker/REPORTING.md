# Reporting Guide

APIRE Compliance Checker generates audit-ready reports.

## Report Types

### 1. Executive Summary
High-level overview for C-suite.
- Overall Compliance Score (0-100%).
- Critical Risks identified.
- Roadmap to compliance.

### 2. Technical Audit
Detailed breakdown for engineers.
- Line-by-line code findings.
- Configuration misconfigurations.
- Remediation steps.

### 3. Framework Specific
- **GDPR Article 30 Record**: Data processing activities.
- **EU AI Act Technical Documentation**: Annex IV requirements.

## Generating Reports

1.  Navigate to **Reports**.
2.  Click **New Report**.
3.  Select **Scope** (Project, Team, or Organization).
4.  Select **Frameworks**.
5.  Click **Generate**.
6.  Download as PDF, HTML, or DOCX.
